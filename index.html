Projeto TCC - Silen

Projeto de TCC, Silen. Silen é um sistema de gerenciamento de entregas. Temos a versão mobile para o entregador e a versão web para o administrador. Explicando basicamente, o administrador tem os cadastros de funcionários e clientes, o mesmo precisa realizar uma entrega para o cliente "x", ele apenas clica em inserir uma nova entrega e um algoritmo insere o melhor motoboy para realizar essa entrega, de acordo com o peso e distância da entrega. Já a versão do motoboy, ao acessar o aplicativo ele tem todas as chamadas abertas, iniciadas e finalizadas dele. O mesmo pode clicar em exibir o mapa e verificar sua localização atual e o percurso até o local da entrega.
O projeto foi feito o com o front-end: HTML5, CSS3, JS utilizando Framework JQuerry. Foi utilizado o node.js para realizar uma migração web para mobile.
Back-end: Java utilizando framework Spring
Banco de dados: SQL utilizando o Workbench
Site: http://silen.herokuapp.com
